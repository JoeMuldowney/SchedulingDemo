# Appointment Scheduling Demo

This is a freelance demo project for a small business appointment scheduling system.  
It allows users to view available time slots and book appointments through a simple interface.

Note: This is a demo project built for freelance portfolio purposes and is not connected to a production payment or user system.

---

## 🚀 Features

- View available appointment slots by date
- Navigate between different days
- Book an appointment with one click
- Real-time slot updates from backend
- Simple confirmation flow after booking
- Responsive UI for desktop and mobile

---

## 🛠️ Tech Stack

- HTML, CSS, JavaScript
- Python (Flask backend)
- REST API for appointments
- Fetch API for frontend communication

---

## 📸 Demo Overview

- Click “Book” on a service
- Select an available time slot
- Confirm booking
- Slot is updated in real time

---

## ⚙️ How to Run Locally

1. Clone the repository:

 - git clone git@github.com:JoeMuldowney/SchedulingDemo.git

2. Install dependencies:
 - pip install -r requirements.txt

3. Run the server:
 - python app.py

4. Enter url in browser:
 - http://127.0.0.1:5000
